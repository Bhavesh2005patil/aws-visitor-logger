# 🚀 AWS Visitor Logger

A professional **serverless visitor tracking application** built using AWS cloud services.

The project automatically captures visitor IP addresses and timestamps whenever a user visits the website and stores the data in DynamoDB using a serverless architecture.

---

## 🌐 Live Architecture

```text
User Browser
     ↓
Amazon EC2
     ↓
API Gateway
     ↓
AWS Lambda
     ↓
Amazon DynamoDB
     ↓
Amazon CloudWatch
```

---

## ☁️ AWS Services Used

| Service | Purpose |
|----------|---------|
| Amazon EC2 | Website Hosting |
| API Gateway | REST API Endpoint |
| AWS Lambda | Serverless Processing |
| DynamoDB | Visitor Data Storage |
| CloudWatch | Monitoring & Logs |
| IAM | Access Management |

---

## ✨ Features

- ✅ Real-time visitor tracking
- ✅ Automatic IP address collection
- ✅ Timestamp logging
- ✅ Serverless backend architecture
- ✅ CloudWatch monitoring
- ✅ Responsive modern UI
- ✅ Production deployment on EC2

## 🔥 Technical Highlights

- Hosted frontend on Amazon EC2 using Apache Web Server.
- Implemented REST APIs using Amazon API Gateway.
- Developed a serverless backend using AWS Lambda.
- Stored visitor information in Amazon DynamoDB.
- Used IAM roles and policies for secure access control.
- Monitored execution using Amazon CloudWatch logs.
