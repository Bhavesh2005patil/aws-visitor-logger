# AWS Visitor Logger

A professional serverless visitor tracking application built using AWS services.

## AWS Services Used

- Amazon EC2
- API Gateway
- AWS Lambda
- Amazon DynamoDB
- IAM
- CloudWatch

## Architecture

Website → API Gateway → Lambda → DynamoDB
