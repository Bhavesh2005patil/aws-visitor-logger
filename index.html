Project Overview

AWS Visitor Logger is a cloud-native serverless application designed to track website visitors in real time.

Every time a visitor opens this website, JavaScript automatically triggers an API request, invoking AWS Lambda which stores visitor details in DynamoDB.

AWS Services Used

Amazon EC2

Hosts the frontend application.

API Gateway

Receives visitor requests.

AWS Lambda

Processes visitor data.

DynamoDB

Stores visitor records.

CloudWatch

Stores logs and metrics.

IAM

Handles permissions securely.
